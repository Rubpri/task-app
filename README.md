# Task App - CRUD

Añade, modifica o borra tareas almacenadas en una base de datos. (Máximo 20 tareas almacenadas). Las tareas actuales se muestran en la página de inicio.

## Stack

- Python/Django/Django REST framework
- JavaScript/React
- Tailwind CSS
- MySQL
